Annotations

Annotations in Java are a feature introduced in JDK5. They are an alternative to adding XML descriptors and marker interfaces. They can be attached to packages, classes, methods, and fields.

To create a custom annotation, you just need to declare a class with public @interface ... Ironically you also have to add the annotations @Retention(RetentionPolicy.RUNTIME) and @Target(Element.Type). The Type as element signifies that the annotation can be applied to a class. FIELD would signify the usage on a parameter and METHOD would apply to method level annotations.
