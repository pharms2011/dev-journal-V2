Unity

What is Unity?

Unity is one of the most powerful gaming engines that also has many applications in real world modeling and can be scripted in C#. It is a free tool to use for personal use to learn and develop portfolio projects, but does cost for developing for profit applications for companies making over 100k a year.

Definitions

Scenes


Scenes are divisions of unity code and compiled assets that form some level of view for the user, menus and levels and such. You can create as many scenes as you would like

Prefabs


Re-usable game elements that have been prefabricated.
