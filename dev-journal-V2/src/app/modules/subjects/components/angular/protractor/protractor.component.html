Protractor

What is Protractor?

Protractor is an e2e testing framework for Angular and Angular JS applications. This means it is used to test that your application will work with all components on your application to make sure it is functioning as desired. These tests usually run in the browser and mock a user or users of you application. Protractor can actually wait for API calls that take time for Angular to load. It's also able to test bindings rather than compiled data which has already been replaced by Angular. Protractor is designed to work with and test Angular directives commonly used, like ngFor and ngIf.

Installing Protractor

To install protractor, just copy and paste the below commands. These will install both protractor and web-driver manager needed to run protractor. Running the final command will start the selenium web driver on port 4444 by default. Selenium will use this port to run commands against your application.

npm install -g protractor
webdriver-manager update
webdriver-manager start
