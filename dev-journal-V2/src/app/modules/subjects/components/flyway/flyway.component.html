Flyway

What is Flyway?

Flyway is an open-sourced database migration tool. Migration can be written in SQL for database-specific syntax or Java for advanced data transformations. It supports various database variants.

How Flyway works

Flyway works on an incremental approach, meaning only a newer version of a schema can be migrated. and only new changes will be applied. By default, this history is kept in a table called SCHEMA_VERSION.
