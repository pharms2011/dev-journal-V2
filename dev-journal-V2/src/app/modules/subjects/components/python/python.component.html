Python

What is Python?

Python is a programming language that is quickly growing in popularity. It is a high level programming language that is known for readability. It is also a dynamically typed language, meaning that type is checked at compile time. It is typically used as an interpreted language, meaning the code is primarily implemented directly, rather than compiling it into machine code first. It is also multi-paradigm as it supports object oriented and structure oriented programming.

pip

pip is a tool used to manage dependencies similar to npm that gathers dependencies from python package index or pipy.

Python 2 vs. Python 3

Python 3 is not backwards compatible. Python2 uses pip, python, and ipython while python 3 uses pip3, python3, and ipython3. Python 3 should be the default choice because Python 2 is considered legacy code.
