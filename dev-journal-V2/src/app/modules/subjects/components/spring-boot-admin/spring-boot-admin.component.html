Spring Cloud Admin

What is Spring Cloud Admin?

Spring Cloud Admin is an application server that allows for the centralization of the health and overall actuator information of all your applications. It was designed by codecentric, but is an open source project. Each application must register itself the server and allow access to their actuator end points.
